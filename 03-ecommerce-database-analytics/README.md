# Ecommerce Database Analytics

> An end-to-end SQL analytics project focused on extracting business insights from e-commerce data.

---

## Project Overview

This project uses a realistic e-commerce database to perform business analysis, reporting, and data-driven decision-making through SQL.

The database includes customers, products, categories, orders, reviews, and transactional data.

---

## Skills Demonstrated

- Multi-Table Joins
- Aggregations
- Revenue Analysis
- Customer Insights
- Product Performance Analysis
- Sales Reporting
- Review Analysis
- Business Intelligence Queries

---

## Database Tables

- Customers
- Categories
- Products
- Orders
- Order_Items
- Reviews

---

## Business Questions Answered

- Who are the top customers?
- Which products generate the highest revenue?
- Which categories perform best?
- What are the monthly sales trends?
- Which products receive the highest ratings?
